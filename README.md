# Amimi Milano — Hub

Pagina hub statica con i link a tutte le dashboard e app operative di Amimi Milano.

**Live**: https://noobhandbag.github.io/amimi-hub/

## Cosa contiene

- **Master Dashboard** — operations a 360° (Shopify, Inventario, Finanza, Marketing, Ads)
- **Finance Dashboard** — Conto Economico e margini
- **Obiettivi Tracker** — roadmap progetti
- **Spedizioni UI** — gestione MyTWS
- **AppSheet Inserisci Acquisto** — form mobile acquisti fornitori
- **Amimi Ads Standalone** — backup dashboard Meta Ads

## Tecnica

- HTML/CSS statico, zero build
- Pubblicato via GitHub Pages dalla root di `main`
- Font: Cormorant Garamond (titoli) + Inter (body)
- Mobile responsive

## Update

Modifica `index.html`, commit + push. GitHub Pages aggiorna entro ~30 secondi.
